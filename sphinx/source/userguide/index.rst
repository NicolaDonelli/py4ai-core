*************

From pypi server

.. code-block:: bash

    pip install py4ai-core

From source

.. code-block:: bash

    git clone https://github.com/NicolaDonelli/py4ai-core
    cd "$(basename "https://github.com/NicolaDonelli/py4AI-core" .git)"
    make install
